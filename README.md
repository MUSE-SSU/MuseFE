# MUSE

### 소개

> MUSE는 하나의 주제에 대해서 사람에 따라 다양하게 표현될 수 있는 창의성에 대해 서로 공유하고 이를 통해 작가들의 영감을 고취하는 창작지원 서비스입니다.

### 개발환경

#### 언어

> -   javascript

#### 프레임워크

> -   React

#### 배포환경

> -   aws EC2

-   NGINX

#### 라이브러리

> -   redux

-   framer-motion
-   gestalt
-   styled-components!
